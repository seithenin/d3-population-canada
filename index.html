Maps Displaying the Population and Growth Rate of Canadian Provinces

Population of Canada by Province from 2012 to 2016

Objective

Determine the densest and least dense provinces of Canada for each year from 2012 to 2016.

Description:

  • Select the year on the right to display the map for that year
  • The colour scale in the top-left is logarithmic

Map Creation Details:

  • Built using D3, a JavaScript library for visualizing data
  • Provincial boundaries (map) generated from Stats Canada dataset
  • Map altered and units converted using QGIS
  • Map file size reduced using Visvalingam-Whyatt line simplification
  • Geographic projection applied to reduce runtime operation costs
  • Population data collected from Stats Canada
  • Population data merged into map data for streamlined retrieval
  • Map converted to TopoJSON and quantized
  • Original dataset size: 54MB, streamlined dataset size: 84KB
  • Logarithmic scale applied to better distinguish population sizes at the lower values

Growth Rate of each Province from 2012 to 2016

Objective

Determine the province with the highest and lowest growth rate from the years 2012 to 2016

Description:

  • The color scale in the top-left is linear since the range is very limited
  • Hovering over each province will reveal the growth rate

Map Creation Details:

  • Map built using the unchanged dataset from the population map
  • Calculations on the dataset were preformed to determine growth rate
  • Map re-generated with growth rate calculations
